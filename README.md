## Sobre este repositorio

Repositorio destinado a hosting de **demos comerciales** de los cursos del catálogo AU Ready v6. Cada subcarpeta es un curso SCORM completo, ejecutable directamente desde el navegador.

Estos cursos están construidos como SCORM 1.2 estándar y funcionan en cualquier LMS compatible (Moodle, Cornerstone, Canvas, etc.). Esta versión web sirve únicamente como **demo de visualización**, sin tracking de progreso (para eso se requiere subir el SCORM al LMS del cliente).
